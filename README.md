## Feature

主な機能は「事前質問の登録と、その自動送信」「回答結果の集計・記録」

- マネージャーはGoogle Authでログインできる
- 質問は初期設定されている
- 質問のカスタマイズができる
- メンバーに回答依頼をメールで送信できる
- メールから回答ページが開ける
- 過去の回答が確認できる
