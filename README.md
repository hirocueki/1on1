# 1 ON 1 をサポートするツール

参考
[1on1ミーティングをより生産的な時間に！部下との対話の価値を高める「Lead Honestly」 \| SELECK \[セレック\]](https://seleck.cc/1127)

## Feature

- 管理者としてログインできる
- 質問は初期設定されている
- 質問のカスタマイズができる
- メンバーの招待ができる
- 「名前」「メールアドレス」「1on1の頻度」「開始日」を登録して、メンバーを招待できる。
- 頻度を選べる。「毎週」「2週ごと」「1ヶ月ごと」
